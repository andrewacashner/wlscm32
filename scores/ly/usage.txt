To include custom macros from ly/phd.ly, 
	put this after \version line:
		\include "../ly/phd.ly" (check relative path)

To produce incipit (see ly/trials/incipit.ly)

		1. Set global incipit data in \IncipitGlobal 
				(\IncipitStyle + meter, key signature)

		2. Put individual-voice data in \IncipitV
				where V is abbrev. for voice, not including instrument name:
				IncipitSi = {
					\IncipitGlobal
					\clef "soprano"
					f'4
				}

		3. In \score, use \InstrumentIncipit after each 
				\new Staff << [here] (or inside Voice definition) to define the 
				long and short instrumentnames with the incipit
					\score {
						<<
						 [...]
						 \new Staff = "Si" 
							<<
							\InstrumentIncipit #"TIPLE 1" #"Ti. 1" ##{\IncipitSi#}
							\new Voice = "EstribilloSi" {
								\MainStyle
								\MusicEstribilloSi
							}
							>>
						>>
						\layout {
							[...]
						}
					}

		(4.) If incipit is not needed (as in coplas score), use \Instrument function to set 
					long and short names:
					
					\new Staff = "Si"
					<<
					\Instrument #"TIPLE I" #"Ti. 1"
					\new Voice = "CoplaSi" {
						\MainStyle
						\MusicCoplaSi
					}
					>>

For ChoirStaff name, use \ChoirStaffName function to set name, 
	inside \new ChoirStaff << [here]:

	\new ChoirStaff
		<<
		\ChoirStaffName #"CORO I"
			\new Staff 
			[...]
		>>
